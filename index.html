Final Project: Weather App

Hello, and welcome to my final project website. This project was done for CSE 383 Web Applications. This page is the landing page. There are two other pages: The Weather Lookup and Search History. These pages can be accessed by the navbar located at the top of each page.

The Weather Lookup page works by entering a city or ZIP Code in the search bar. The search query is then used in an API call to the TomTom Search API Fuzzy Search to get the latitude and longitude of that location. This latitude and longitude is then used to get the weather data from the Open Weather API and the weather for the next five days are displayed on the page. This data is then saved in an SQL Database for use on the Search History page.

The Search History page works by entering a calendar date and the number of desired results. The location, latitude, and longitude for the searches made on that day are then displayed. The button for that search can be clicked to display the full weather data just like on the Weather Lookup page.
